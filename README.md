# Run in Development

1. Clone the repository
2. Run:
   ```bash
   npm install
   ```
3. Make sure Nest CLI is installed:

   ```bash
   npm i -g @nestjs/cli
   ```

4. Rename the file .env.template:

   ```bash
   .env
   ```

5. Start the database:
   ```bash
   docker-compose up -d
   ```

## Stack Used

- MongoDB
- NestJS
